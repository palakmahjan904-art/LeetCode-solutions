## 🧠 Key Takeaways From This Series

* **Avoiding Nested Loops:** Transitioned from brute-force O(n^2) to linear O(n) by utilizing the sliding action (adding new element, dropping the oldest).

* **Mathematical Shortcuts:** Replaced continuous float division with a pre-calculated target sum (threshold * k) inside LeetCode 1343 to boost speed.

* **Space Optimization:** Achieved top-tier memory ranks by avoiding auxiliary vectors and maintaining states inside simple variables.
